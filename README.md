# Plant Care Pro

Профессиональная веб-система для ухода за комнатными растениями с интуитивно понятным интерфейсом и мощными возможностями отслеживания.

## ✨ Основные возможности

- ✅ Добавление растений с индивидуальными настройками
- ✅ Настраиваемые интервалы: полив, подкормка, пересадка
- ✅ Автоматические напоминания
- ✅ Отметка выполненных задач
- ✅ Экспорт данных в JSON
- ✅ Адаптивный дизайн

## 🚀 Быстрый старт

1. Откройте `index.html` в браузере
2. Добавьте свои растения
3. Получайте напоминания и ухаживайте

## 🛠️ Технологии

- HTML5
- CSS3  
- JavaScript ES6+
- LocalStorage
